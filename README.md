# University United Front Information Management System
这是一个基于SpringBoot的高校统战系统，为学生做统战的宣传工作。有首页、新闻分页展示、建言献策模块、意见反馈模块、用户管理、新闻上传等功能模块
技术栈：SpringBoot+Vue+Mybatis-Plus+Element-ui
